## Purpose

This release adds TypeScript Intent Registry / Public Key Binding.

It verifies that the public key used in a detached intent signature is present in a declared local registry and marked active.

## Scope

Supported:

- local registry JSON parsing,
- registry entry lookup,
- signer label matching,
- public key hash matching,
- public key matching when declared,
- active/valid/trusted/enabled status check,
- human-readable and JSON output.

Not supported:

- signed registry verification,
- registry hash anchoring,
- revocation ledger validation,
- signer legal authority,
- organizational role validation,
- policy compliance.

## Security

The registry is an input artifact.

v2.12.2 proves consistency with the provided registry file only.

It does not prove legal identity, signer authority, role validity, or real-world truth.
