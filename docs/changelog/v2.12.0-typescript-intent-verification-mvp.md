## Purpose

This release adds the first TypeScript Proof of Intent verification MVP.

The MVP verifies that an intent attestation declares a record hash that matches the target DELTA record.

## Scope

Supported:

- intent JSON readability,
- record JSON/file readability,
- declared record hash extraction,
- record hash recomputation,
- record hash binding verification,
- human-readable CLI output,
- JSON CLI output using the v2.10.2 contract style.

Not supported:

- detached intent signature cryptographic verification,
- legal approval,
- signer authority,
- real-world identity,
- registry trust,
- policy compliance,
- wallet ownership.

## Security

This release verifies record hash binding only.

It must not be described as full Proof of Intent authority verification.
