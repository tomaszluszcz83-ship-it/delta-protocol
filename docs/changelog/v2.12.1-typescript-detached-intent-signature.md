## Purpose

This release extends TypeScript Proof of Intent verification with detached Ed25519 intent signature verification.

## Scope

Supported:

- record hash binding from v2.12.0,
- detached intent signature JSON parsing,
- intent hash binding,
- `signature_body_hash` self-check,
- public key hash check,
- Ed25519 detached signature verification,
- human-readable and JSON CLI output.

Not supported:

- registry trust,
- signer authority,
- role validity,
- policy compliance,
- legal approval,
- real-world identity.

## Security

This release verifies that an Ed25519 key signed a canonical signature body bound to the target intent attestation.

It does not prove that the signer is legally authorized or that any policy was satisfied.
