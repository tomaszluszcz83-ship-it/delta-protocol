## Purpose

This release adds TypeScript verification for detached signed `.delta` bundles.

## Supported

- bundle integrity check via the v2.9.3 verifier,
- exact bundle file SHA-256 check,
- bundle hash binding,
- signature body hash self-check when declared,
- public key hash check when declared,
- Ed25519 signature verification.

## Security

This release verifies signed bundle binding only.

It does not prove legal truth, real-world truth, identity, authority, wallet ownership, compliance, trust validity, or contained proof correctness.
