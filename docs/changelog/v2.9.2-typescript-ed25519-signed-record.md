## Purpose

This release adds the first TypeScript Ed25519 signature verification layer.

## Supported

- Ed25519 verification for the v2.9.2 TypeScript signed record MVP profile,
- public key hash check,
- record hash recomputation,
- signed record demo generation,
- signed record CLI verification.

## Security

This is a narrow MVP signature profile.

It does not prove legal truth, real-world truth, signer authority, identity, wallet ownership, compliance, or trust validity.

It does not verify replay, intent, audit, publication, trust, wallet, bundle, or signed bundle proofs.
