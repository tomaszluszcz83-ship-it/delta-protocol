## Purpose

This release introduces a minimal TypeScript verifier for DELTA L0/L1.

The strategic purpose is to begin moving DELTA from a Python-only reference implementation toward a multi-implementation protocol.

## Scope

Supported:

- canonical JSON vector verification,
- SHA-256 verification,
- duplicate key rejection in strict JSON text parsing,
- float/NaN/Infinity/unsafe integer rejection,
- basic record required field check,
- basic record hash recomputation.

Not supported:

- schemas,
- signatures,
- replay,
- intent,
- audit,
- publication,
- trust,
- wallet,
- bundle,
- signed bundle.


## Security boundary

This is an experimental verifier for canonical JSON and basic hashes only.

It does not prove legal truth, real-world truth, identity, authority, wallet ownership, compliance, or trust validity.
