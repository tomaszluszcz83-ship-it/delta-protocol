## Purpose

This release adds JSON Schema pre-verification to the TypeScript verifier.

## Supported

- repository-local `schemas/` loading,
- AJV 2020 schema compilation,
- named schema validation,
- schema registry compile checks.

## Security

Schema validation is pre-verification only.

It does not prove cryptographic validity, legal truth, real-world truth, identity, authority, wallet ownership, compliance, replay correctness, or trust validity.

## Tested

Expected checks:

```powershell
cd verifier\ts
npm install
npm run build
npm run verify-vectors
npm run verify-schemas

cd ..\..
python src/delta_cli.py verify-all
python tools\delta_canonical_json.py verify-vectors --vectors tests\vectors\canonical-json\vectors.json

```
