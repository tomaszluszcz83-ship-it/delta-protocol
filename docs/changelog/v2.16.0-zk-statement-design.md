## Purpose

This release defines the first DELTA ZK statement model.

It classifies public inputs and private witnesses, documents limitations, and identifies candidate circuits for a future v3.0.0-alpha ZK proof-of-concept.

## Scope

Documented:

- core ZK statement,
- public inputs,
- private witness,
- non-goals,
- candidate circuits,
- privacy leakage risks,
- ZK roadmap.

## Security

This release implements no ZK code.

It does not add a circuit, prover, verifier, trusted setup, or production ZK claim.

It is a design and anti-overclaiming milestone.
