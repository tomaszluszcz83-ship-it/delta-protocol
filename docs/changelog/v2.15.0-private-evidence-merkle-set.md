# DELTA v2.15.0 — Private Evidence Package / Merkle Evidence Set Changelog Entry

Type: Private evidence / ZK preparation milestone

## Added

- `tools/delta_private_evidence_set.py`
- `docs/private-evidence/private-evidence-merkle-set-v2.15.0.md`

## Purpose

This release extends the private evidence commitment layer from one evidence item to a Merkle set of multiple private evidence commitments.

It prepares DELTA for ZK statement design by creating a clean public root / private witness structure.

## Scope

Supported:

- public Merkle evidence set package,
- private opening package,
- salted SHA-256 commitments per evidence item,
- Merkle leaf hashes,
- Merkle root,
- per-entry Merkle proofs,
- public package verification,
- selective disclosure verification,
- tampered evidence rejection.

## Security

This release is not encryption and not ZK.

The public package contains commitments, leaf hashes, and a Merkle root only.

The private opening must not be committed publicly.

Disclosure verification proves matching evidence/opening/commitment/Merkle root only.

It does not prove evidence truth, completeness outside the committed set, legal validity, policy satisfaction, or regulatory compliance.
