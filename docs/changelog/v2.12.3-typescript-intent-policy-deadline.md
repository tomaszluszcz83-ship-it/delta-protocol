## Purpose

This release adds TypeScript Intent Policy / Deadline checks.

It verifies that an intent references a local policy and that the effective deadline has not expired at the declared verification time.

## Scope

Supported:

- local policy JSON parsing,
- intent policy ID matching,
- policy status check,
- deadline parsing,
- deadline check using `--now`,
- human-readable and JSON output.

Not supported:

- signed policy artifacts,
- policy registry,
- revocation ledger validation,
- legal approval,
- regulatory compliance.

## Security

The policy file is an input artifact.

v2.12.3 proves consistency with the provided policy file under the declared time only.

It does not prove that the policy itself is legally valid, globally trusted, or sufficient for compliance.
