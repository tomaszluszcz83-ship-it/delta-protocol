# DELTA v2.8.3 — Replay Environment Checker Changelog Entry

Type: MVP tool release

## Added

- `tools/delta_replay_check_env.py`
- `docs/replay/delta-replay-check-env-v2.8.3.md`

## Purpose

This release adds a practical local checker for replay environment declarations.

It bridges v2.8.2 documentation with executable behavior.

## Supported in v2.8.3

- OS family check
- Python version check
- selected Python package version check
- `MATCH`
- `MISMATCH`
- `MANUAL_REVIEW_REQUIRED`

## Security

- Does not prove the original execution environment.
- Does not verify Docker, Nix, Guix, hardware attestation, or network state.
- Unsupported fields require manual review.
- Does not replace Proof of Replay, Proof of Intent, Proof of Audit, Proof of Trust, or revocation checks.
