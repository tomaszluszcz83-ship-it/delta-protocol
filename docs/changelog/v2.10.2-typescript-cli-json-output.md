- `docs/typescript-cli-json-output-v2.10.2.md`

## Purpose

This release adds machine-readable JSON output wrappers for selected TypeScript verifier commands.

It improves CI/CD and automation usability without changing verifier cryptography.

## Exit codes

| Exit code | Code name |
| --- | --- |
| 0 | `OK` |
| 1 | `VERIFICATION_FAILED` |
| 2 | `USAGE_ERROR` |
| 3 | `INTERNAL_ERROR` |

## Scope

Implemented JSON wrappers:

- signed record verification,
- public `.delta` bundle verification,
- signed `.delta` bundle verification.

## Security

This release adds no new cryptographic functionality.

It changes output format and exit-code discipline only.
