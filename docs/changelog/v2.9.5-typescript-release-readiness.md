## Purpose

This release documents the TypeScript verifier capability matrix after v2.9.0–v2.9.4.

It clarifies:

- what TypeScript verifies today,
- what TypeScript does not verify,
- how TypeScript compares with the Python Alpha Reference Implementation,
- the boundaries of bundle and signed bundle verification,
- what public claims are safe,
- what public claims must be avoided.

## Security

This release adds no new cryptographic functionality.

It reduces overclaiming risk by documenting the exact boundary of the TypeScript verifier.
