## Purpose

This release adds TypeScript verification for public `.delta` ZIP bundles.

## Supported

- required manifest check,
- duplicate filename rejection,
- path traversal rejection,
- sensitive filename rejection,
- artifact SHA-256 verification,
- artifact size verification,
- unreferenced artifact rejection.

## Security

This verifier checks the bundle container and manifest only.

It does not verify signed bundle signatures or contained proof-specific cryptographic validity.

It does not prove legal truth, real-world truth, identity, authority, wallet ownership, compliance, or trust validity.

## Tested

cd verifier\ts
npm install
npm run build
npm run verify-vectors
npm run verify-schemas
npm run verify-bundle -- --bundle ..\..\.delta\bundle-tests\B-293\sample.delta

python src/delta_cli.py verify-all
