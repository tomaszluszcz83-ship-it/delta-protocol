# DELTA v2.7.3 — Documentation Refresh Changelog Entry

Type: Documentation-only release

## Added

- Documentation refresh for the v2.7.x UX/adoption milestones.
- Reviewer-facing workflow for:
  - `delta_export.py`
  - `delta_intent_create.py`
  - `delta_bundle.py`
- Updated explanation of the public artifact flow:
  - DELTA record
  - unsigned intent draft
  - Markdown/HTML report
  - portable `.delta` bundle
- Clarified relationship between v2.7.2 Portable Bundle and future v2.8.0 Signed Bundle.

## Security

- Reconfirmed that UX tools do not create new cryptographic proofs.
- Reconfirmed that `.delta` bundles must not contain private keys, secrets, raw private evidence, or decrypted evidence.
- Reconfirmed that bundle verification does not replace proof-specific DELTA verifiers.
- Reconfirmed that DELTA does not prove legal truth, real-world truth, identity, wallet balance, or regulatory compliance by itself.
