# DELTA v2.14.0 — Private Evidence Commitment Profile Changelog Entry

Type: Private evidence / ZK preparation milestone

## Added

- `tools/delta_private_evidence_commitment.py`
- `docs/private-evidence/private-evidence-commitment-v2.14.0.md`

## Purpose

This release adds a Private Evidence Commitment Profile.

It creates a public commitment to private evidence and a private opening file for later disclosure.

## Scope

Supported:

- create public commitment package,
- create private opening package,
- verify public package self-check,
- verify disclosed evidence against public commitment and private opening,
- salted SHA-256 evidence commitments,
- explicit anti-overclaiming security boundaries.

## Security

This release is not encryption and not ZK.

The public package does not reveal raw evidence or salt.

The private opening must not be committed publicly.

Disclosure verification proves matching evidence/opening/commitment only.

It does not prove evidence truth, completeness, legal validity, policy satisfaction, or regulatory compliance.
