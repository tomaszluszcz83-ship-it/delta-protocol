Security model

  • This page has no backend endpoint and no upload action.
  • The application code does not call fetch, XMLHttpRequest, analytics, forms, cookies, or beacons.
  • Hashes are computed over deterministic canonical JSON bytes.
  • Ed25519 signatures are verified over canonical JSON bytes, not over prehashed bytes.
  • For production hardening, vendor and pin the JS dependencies locally under docs/vendor/.
