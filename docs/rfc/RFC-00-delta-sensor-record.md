## 1. Abstract

This RFC defines the first draft of the DELTA Sensor Record.

A DELTA Sensor Record is a signed, hash-committed record generated by an automated system that observes a change, executes a measurement method, records the result, commits to private evidence, and signs the resulting record body.

This RFC covers the sensor layer only.

It does not define the full DELTA-0 bundle:
